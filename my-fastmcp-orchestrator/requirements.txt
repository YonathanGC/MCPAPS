# === General Notes ===
# - To install all uncommented dependencies:
#   pip install -r requirements.txt
#
# - For development, consider using a virtual environment:
#   python -m venv venv
#   source venv/bin/activate  # macOS/Linux
#   .\venv\Scripts\activate   # Windows
#
# - Some libraries (especially for Autodesk products) may require the host application
#   to be installed on the same machine where this orchestrator runs.
#
# - Always check the documentation for each library for specific version compatibility
#   with your host applications (e.g., AutoCAD version, Revit version).
#
# - If you encounter issues installing geospatial libraries like GDAL, Fiona, Shapely, etc.,
#   they often have binary dependencies. Using pre-compiled wheels from sources like
#   Christoph Gohlke's Python Extension Packages for Windows (https://www.lfd.uci.edu/~gohlke/pythonlibs/)
#   or package managers like Conda (conda install -c conda-forge gdal) can be helpful.
